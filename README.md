# Sketch3D Web

Sketch3D Web is a React-based application that allows users to create and visualize shapes in a simple web interface. The project was built using **Create React App** and demonstrates basic frontend development using React.

## Features

* Interactive shape creation
* Real-time updates in the browser
* Simple and responsive UI

## Installation

1. Clone the repository

```
git clone https://github.com/Omkar090804/Sketch3D_Web.git
```

2. Navigate to the project folder

```
cd Sketch3D_Web
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm start
```

The app will run at:
http://localhost:3000

## Build for Production

```
npm run build
```

This creates an optimized production build in the **build** folder.
